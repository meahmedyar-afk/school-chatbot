# School Information FAQ Chatbot

A rule-based chatbot built using Python that answers common school-related questions such as admissions, fees, subjects, timings, and contact details.

## Features
- Keyword-based intent detection
- Continuous conversation loop
- User-friendly responses
- Handles unknown user inputs

## Technologies Used
- Python
- Conditional Logic

## How to Run
1. Install Python
2. Download or clone this repository
3. Run the chatbot using:
   py school_chatbot.py

## Use Case
This chatbot can be used for school websites or help desks to quickly answer common student and parent queries.
